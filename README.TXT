DPTransport-v2 es la continuación de DPTransport, un proyecto de introducción de Java que ejecuta una simulación fictícea de redes de taxis y clientes.  
Esta es una versión mejorada que cuenta con diversas extensiones y mejoras de calidad de código de todas las categorías mencionada por el profesorado.
Creado para la asignatura de Desarrollo de Programas del Grado de Ingeniería en Ingeniería del Software en la UEx año 2023/2024.  

Proyecto hecho por:
    - Manuel Alonso González.
    - David de Miguel Palomino.
    - José Manuel de Torres Domínguez.

Para ejecutar el programa (main()), ejecutar el main que hay en la clase TaxiDemoFinal.

Nuestro proyecto cuenta con los siguientes extras:

    - Todas las extensiones, de todas las calidades.
